 *    - 计算：成型余量 = 硫化余量 - 胎胚库存
 *    - 判断：能否在收尾日前完成？
 * 3. 延误量追赶：
 *    - 如果做不完，计算延误量，平摊到未来3天
 *    - 检查未来3天满产是否能追上
 * 4. 满产判断：
 *    - 如果未来3天满产仍追不上，通知月计划调整（调用硫化接口）
 *
 * @author APS Team
 */
