## 二、与APS成型排程系统的关系

### 2.1 数据流向图

```
┌─────────────────────────────────────────────────────────────────┐
│                    月度生产计划定稿                              │
│          (FactoryMonthPlanProductionFinalResult)                │
│                     T_MP_MONTH_PLAN_PROD_FINAL                  │
└──────────────────────────┬──────────────────────────────────────┘
                           │
                           ▼ 按日期拆分
┌─────────────────────────────────────────────────────────────────┐
│                      硫化日计划                                  │
│                  (VulcanizingPlan)                              │
│                   T_CX_VULCANIZING_PLAN                         │
│   字段映射:                                                      │
│   material_code ← materialCode                                  │
│   plan_quantity ← dayX (当日计划量)                              │
│   priority ← 根据productionType确定                             │
└──────────────────────────┬──────────────────────────────────────┘
                           │
                           ▼ APS排程算法
┌─────────────────────────────────────────────────────────────────┐
│                      排程主表                                    │
│                   (ScheduleMain)                                │
│                   T_CX_SCHEDULE_MAIN                            │
│   字段映射:                                                      │
│   schedule_date ← 计划日期                                       │
│   total_quantity ← 当日总排产量                                  │
└──────────────────────────┬──────────────────────────────────────┘
                           │
                           ▼
┌─────────────────────────────────────────────────────────────────┐
│                      排程明细表                                  │
│                   (ScheduleDetail)                              │
│                   T_CX_SCHEDULE_DETAIL                          │
│   字段映射:                                                      │
│   material_code ← materialCode                                  │
│   plan_quantity ← 分配到机台的量                                 │
│   machine_code ← cxMachineCode (拆分)                           │
│   sequence ← productionSequence                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 2.2 核心字段映射关系

| 月计划字段 | APS表 | APS字段 | 转换逻辑 |
|-----------|-------|---------|---------|
| materialCode | t_cx_material | material_code | 直接映射 |
| structureName | t_cx_material | product_structure | 直接映射 |
| mainPattern | t_cx_material | main_pattern | 直接映射 |
| pattern | t_cx_material | pattern | 直接映射 |
| dayVulcanizationQty | t_cx_stock | - | 计算硫化能力 |
| cxMachineCode | t_cx_schedule_detail | machine_code | 拆分到多机台 |
| curingTime | t_cx_material | vulcanize_time_minutes | 直接映射 |
| day1~day31 | t_cx_vulcanizing_plan | plan_quantity | 按日期拆分 |

### 2.3 业务流程集成点

1. **月计划导入**：从T_MP_MONTH_PLAN_PROD_FINAL读取月度计划
2. **日计划拆分**：将day1~day31拆分为每日硫化计划
3. **排程生成**：调用APS核心算法生成日排程
4. **结果回写**：将排程结果更新到月计划的排产量字段

## 三、集成方案设计

### 3.1 新增数据表

```sql
-- 月度生产计划定稿表（与现有系统对接）
T_MP_MONTH_PLAN_PROD_FINAL
```

### 3.2 新增服务

```java
// 月计划服务接口
MonthPlanService
├── importMonthPlan()          // 导入月计划
├── splitToDailyPlan()         // 拆分为日计划
├── generateDailySchedule()    // 生成日排程
└── syncScheduleResult()       // 同步排程结果
```

### 3.3 集成流程

```
月计划定稿 → 日计划拆分 → APS排程 → 结果同步 → 发布执行
```
